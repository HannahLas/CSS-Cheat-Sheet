CSS Cheatsheet

Need help remembering CSS code? Use this cheetsheet to automatically select the correct code that you can can copy and paste into your CSS.

CSS Flexbox Containers

To create a flexbox container (rows is the default property), use the following code:

 
    .row {
        display:flex;
        flex-wrap:wrap;
    }

Responsive Flexbox

To make FlexBox responsive you can simply use this code to stack the containers on top of each other:

    @media (max-width: 800px) {
        .row {
          flex-direction: column;
        flex-wrap: wrap;
          align-content: center;
        } 
    }

CSS Grid

Build a 9 by 9 grid with the following code: 

    .grid {
        display:grid;
        width:100%
        grid-template-columns: repeat 3 (1r)
        grid-template-rows: repeat 3 (1fr)
    }

Select Text

Use this code to select all of the text within a section you click on. 

 
    pre {
        -webkit-user-select: all;
        user-select: all;
      }

Linear Gradient

This will create the linear gradient shown on this page:

        column {
        background: linear-gradient(45deg, 
        paleturquoise, pink, skyblue);
        }
